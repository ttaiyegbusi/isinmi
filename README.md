# Isinmi — Rest. Healing. Refuge.

A landing page for **Isinmi**, a community providing rest, peace, and support for survivors of sexual abuse in Nigeria and Africa.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite 5**
- **Tailwind CSS v3**
- **Framer Motion**

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import the repository in [Vercel](https://vercel.com)
3. Vercel auto-detects Vite — no config needed
4. Hit Deploy

## Project Structure

```
src/
  components/
    Nav.tsx           # Floating pill navigation
    Hero.tsx          # Full-screen hero with wordmark
    AboutUs.tsx       # Two-column editorial about section
    TheData.tsx       # Data visualization with ascending pillars
    MissionStatement.tsx  # "At Isinmi, We are that Refuge."
    OurReach.tsx      # Reach section with country list
    OurValues.tsx     # Values grid on dark teal
    Testimonials.tsx  # Testimonial cards carousel
    Footer.tsx        # CTA + giant ISINMI wordmark
```
